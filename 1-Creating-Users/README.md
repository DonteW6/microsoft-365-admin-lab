# Creating Users in Microsoft 365

## Objective

Learn how to create user accounts in Microsoft 365 Admin Center.

## Environment

- Microsoft 365 Business Premium Trial
- Microsoft Entra ID
- Administrator Account

## Steps Performed

1. Logged into Microsoft 365 Admin Center.
2. Navigated to Users → Active Users.
3. Selected Add User.
4. Created five employee accounts.
5. Assigned Microsoft 365 Business Premium licenses.
6. Generated temporary passwords.
7. Required password change at first sign-in.

## Skills Demonstrated

- User account provisioning
- License assignment
- Identity management
- Microsoft 365 administration

## What I Learned

Creating users in Microsoft 365 involves assigning usernames, licenses, and temporary passwords. Administrators can enforce password changes at first login to improve security.
