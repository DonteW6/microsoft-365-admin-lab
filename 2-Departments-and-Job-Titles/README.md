# Assigning Departments and Job Titles

## Objective

Learn how to organize users by assigning departments and job titles in Microsoft 365 Admin Center to better simulate a real-world business environment.

## Environment

- Microsoft 365 Business Premium Trial
- Microsoft 365 Admin Center
- Microsoft Entra ID
- Administrator Account

## Steps Performed

1. Logged into Microsoft 365 Admin Center.
2. Navigated to **Users → Active users**.
3. Selected each employee account individually.
4. Opened the user's profile.
5. Updated the **Department** field.
6. Assigned an appropriate **Job Title** based on the employee's role.
7. Saved the changes for all employees.

## Skills Demonstrated

- User profile management
- Organizational administration
- Department and role assignment
- Microsoft 365 user administration
- Identity management

## What I Learned

Assigning departments and job titles improves user organization within Microsoft 365 and helps reflect an organization's structure. This information can be used for reporting, directory searches, administrative management, and future identity and access management (IAM) tasks.
