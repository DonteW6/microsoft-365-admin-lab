# Password and Identity Management

## Objective

Learn how to manage user identities in Microsoft 365 by performing common IT help desk and identity lifecycle tasks, including password resets and employee offboarding.

## Environment

- Microsoft 365 Business Premium Trial
- Microsoft 365 Admin Center
- Microsoft Entra ID
- Administrator Account

## Steps Performed

### Password Reset

1. Logged into Microsoft 365 Admin Center.
2. Navigated to **Users → Active users**.
3. Selected the user account for **Lionel Messi**.
4. Reset the user's password by generating a temporary password.
5. Required the user to change their password at the next sign-in.
6. Verified that the password reset completed successfully.

### Employee Offboarding

1. Selected the user account for **Tom Brady**.
2. Blocked the user's ability to sign in.
3. Removed the Microsoft 365 Business Premium license.
4. Removed the user from assigned security groups.
5. Transferred ownership of the **Sales-Team** security group from **Tom Brady** to **Lionel Messi**.
6. Verified that organizational access had been revoked while preserving the user account.

## Skills Demonstrated

- Password administration
- User account management
- Identity lifecycle management
- Microsoft Entra ID administration
- License management
- Security group administration
- Group ownership management
- Identity and Access Management (IAM)

## What I Learned

Managing user identities extends beyond creating accounts. Password resets, account access management, license removal, security group updates, and ownership transfers are essential administrative tasks that help maintain security and ensure business continuity. Proper offboarding procedures reduce the risk of unauthorized access while preserving organizational resources and access control.
