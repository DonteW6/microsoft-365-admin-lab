# Microsoft Entra ID Administration

## Objective

Learn how to navigate Microsoft Entra ID and understand administrative roles used to securely manage user identities and permissions.

## Environment

- Microsoft 365 Business Premium Trial
- Microsoft Entra Admin Center
- Administrator Account

## Steps Performed

1. Logged into Microsoft Entra Admin Center.
2. Navigated to Identity → Users → All users.
3. Reviewed user properties including departments, job titles, managers, and group memberships.
4. Explored built-in administrative roles.
5. Reviewed the permissions associated with Global Administrator, User Administrator, and Helpdesk Administrator.

## Skills Demonstrated

- Microsoft Entra ID administration
- User identity management
- Administrative role management
- Role-Based Access Control (RBAC)
- Principle of least privilege
- Identity and Access Management (IAM)

## What I Learned

Microsoft Entra ID uses role-based access control to delegate administrative responsibilities while limiting unnecessary privileges. Assigning specific roles, such as Helpdesk Administrator, allows organizations to securely distribute administrative tasks without granting full Global Administrator access.
