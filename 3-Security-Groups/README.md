# Creating Security Groups

## Objective

Learn how to create and manage security groups in Microsoft Entra ID to organize users and simplify access management within a Microsoft 365 environment.

## Environment

- Microsoft 365 Business Premium Trial
- Microsoft 365 Admin Center
- Microsoft Entra ID
- Administrator Account

## Steps Performed

1. Logged into Microsoft 365 Admin Center.
2. Navigated to **Teams & groups → Active teams & groups**.
3. Selected **Add a group**.
4. Chose **Security** as the group type.
5. Created security groups for each department.
6. Assigned an owner to each security group.
7. Added the appropriate users as members based on their department and role.
8. Verified group membership and saved all changes.

## Skills Demonstrated

- Microsoft Entra ID administration
- Security group creation
- Group ownership management
- User membership management
- Role-based access control (RBAC)
- Identity and Access Management (IAM)

## What I Learned

Security groups provide an efficient way to manage user permissions by assigning access to groups instead of individual users. Organizing employees into department-based security groups simplifies administration, improves scalability, and supports role-based access control (RBAC), which is a common practice in enterprise IT environments.
